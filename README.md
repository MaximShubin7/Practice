Интернет-магазин на React
📖 Описание проекта
Веб-приложение интернет-магазина, разработанное в рамках учебной практики. Проект демонстрирует ключевые принципы реактивных веб-приложений с использованием современного стека технологий.

🚀 Основной функционал
Каталог товаров — просмотр списка товаров с поиском в реальном времени

Карточка товара — детальная информация о товаре (описание, характеристики, цена)

Корзина — добавление/удаление товаров, изменение количества, подсчет итоговой суммы

Избранное — сохранение понравившихся товаров

Авторизация — регистрация и вход с JWT-токенами

Разделение ролей — пользователь и администратор

Админ-панель — управление товарами (добавление, удаление)

🛠 Технологии
Frontend: React 19, TypeScript, Vite, React Router, React Hook Form + Yup, SCSS Modules

Backend: JSON Server, JWT

Для запуска проекта введите в терминал 🚀:
npm install
npm run build
npm run start
